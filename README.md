# Sword & Supper New Items Viewer

An unofficial viewer for equipment introduced in the latest
Sword & Supper release.

## Information included

- Item icons and descriptions
- Stats for every level
- Ultimate abilities and exact parameters
- Required player levels
- Upgrade material requirements
- Farming and drop sources
- Conditional loot-pool probabilities

## Viewer

Open the GitHub Pages deployment for this repository.

## Disclaimer

This is an unofficial community project and is not affiliated with
Sword & Supper, Reddit, or the game's developers.

Item data was extracted from the publicly delivered game client.
Images and fonts are loaded from the live game asset server.

Drop percentages represent conditional probabilities within the
identified loot pool. Actual per-mission probabilities may be lower
because of additional event or wrapper probabilities.
